Banner Manager v1.0 - General Overview

Developed by Wiseguy Digital :: http://www.wiseguydigital.com

This plug-in allows you to add a banner in either an image format (usually gif, jpeg or png) or as an Adobe® Flash .swf file. If using the .swf format for a banner, you will firstly need to permit the use of Flash banners in the Banner Manager configuration file (bannerConfig.php). There is a master .swf file in the Banner Manager plug-in that sets up the link to work with the tracker, so the use of getURL() in ActionScript will not work.

Banner Overview page

This page simply gives an overview of the banners and shows you the number of page displays, clicks on that banner and hence the Click Through Rate (CTR) as a percentage.

You can click on the banner file to view it. You also have the option to add, edit or delete a banner from this page. If a banner has been associated to a particular page then you may also click on the link to view the page.

Add or edit a banner pages

This page contains a form for uploading your banners as listed below:

Campaign name

This is a name for the banner for internal reference and it will also be shown as "alt" text on image banners.

Banner file

Use the browse button to locate the file on your computer's hard drive.

Banner position

This selects which placeholder in the template file to associate this banner with. These are defined in the Banner Manager configuration file (bannerConfig.php).

Link URL

This is the full URL of the page that you wish the user to be redirected to after they click on the banner e.g. http://www.tradingeye.com.

Banner type

If you have set the option to allow SWFs in the Banner Manager configuration file (bannerConfig.php), then you will see this section to choose whether you wish to have a Flash banner or an image banner.

Banner size

There are 2 fields here that must be entered: the banner's width and height in pixels.

Display page

If you wish for the banner to appear on a particular page in the site, simply enter the URL (in full), otherwise the banner will be shown across the site.

Banner is active?

This simply allows you to activate or de-activate a banner without having to delete it permanently.
