Help File: Sage CSV's [ X ]

The CSV's below can be downloaded and then imported manualy into SAGE.
There are two CSV's

  • Customer CSV
  • Transaction (order) CSV

Customer CSV Download: Account Ref format should be a fixed style e.g. AB<ID> or ABCD<ID>
The Account Ref should contain the first X letters of your company/surname.
ID could be any numeric letters.

Transaction (order) CSV: This file holds information related to the orders placed at the website. You can filter the information based on the date any particular order was placed.
Important note: The Sage CSV import doesn't support importing orders or invoices you can only import raw transactions.
There are 2 transactions which relate to a Sales Invoice (SI) and a Payment on Account(SA).

  • Sale Invoice: Nominal Code is required - 4000 is default value
  • Payment on Account (Optional): You need to tick a check box for "Posts Sales Receipt for every Invoices" and "post Receipts To" is required.
