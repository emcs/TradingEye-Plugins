------------------------------------------------------------------------
                    Instructions for installation
------------------------------------------------------------------------


1. Create a new directory called "sagecsv" on your server in the "plugins" directory.

2. Unzip the file sagecsv.zip and upload the contents to this new directory.

3. Copy the javascript prototype.js into "jscript" folder ( Ex: www.yoursite.com/jscripts/)

4. Log in to the shop's administration section and choose "Plugin Modules" from the "Admin" box on the left-hand side.

5. Select "Add new application" (the link in the top right of the table).

6. Enter the following details and then click on "Add application":

	- Application name: "Sage CSV"
	- Directory name: "sagecsv"
	- Version of application: 1.0
	- Help file name: "sagehelp.htm"
	- Security setting: Administrator
	- Description: "Sage plugin for Sage 50 Accounts 2008"
	- On/Off: tick this checkbox

7. On the left-hand side of the screen should now be a new sub-menu called Modules and a new link called "Sage CSV"

8. Click on the "Sage CSV" link. For the help about Sage plugin click on the question mark icon in the top left of the table.

9. That's it!
