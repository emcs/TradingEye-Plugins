Help File: QuickBooks CSV's [ X ]

The CSV's below can be downloaded and then imported manually into QuickBooks.
There are two CSV's

  • Customer CSV
  • Supplier CSV

Customer CSV Download: In order to import your customer CSV correctly you must first create a customer mapping within Quickbooks.

Within QuickBooks, in the customer centre click on excel and then import from excel. And then click on advanced import.

You then need to browse to the customer CSV file you wish to import. You then need to create a new customer mapping.
From the dropdown box select ADD NEW.

The following fields need to be mapped using the headings in the CSV file:

Job or Customer Name
Company Name
First Name
Last Name
Contact
Phone
Email
Billing Address 1
Billing Address 2
Billing Address 3
Billing Address 4
Billing Address 5
Account Number

Once you have mapped all the fields click save and then import.

Supplier CSV: This is the same process is similar to the Customer import.
Within QuickBooks, in the vendor centre click on excel and then import from excel. And then click on advanced import.

You then need to browse to the supplier CSV file you wish to import. You then need to create a new vendor mapping.
From the dropdown box select ADD NEW.

The following fields need to be mapped using the headings in the CSV file:

Name
Company Name
Address 1
Address 2
Address 3
Address 4
Address 5
Phone
Email
Account Reference

Once you have mapped all the fields click save and then import.
